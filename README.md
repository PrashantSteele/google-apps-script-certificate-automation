# Google Apps Script Certificate Automation

Automate bulk certificate generation using:

- Google Sheets
- Google Slides Template
- Google Drive
- Gmail

## Features

✅ Bulk certificate generation  
✅ Personalized names  
✅ Auto PDF creation  
✅ Auto email delivery  
✅ Save certificate URL in sheet  
✅ Date auto insertion  
✅ Google Slides template support  

## Technologies Used

- Google Apps Script
- Google Sheets API
- Google Slides API
- GmailApp
- DriveApp

## Setup

1. Create Google Sheet
2. Add columns:

| Name | Email | Certificate URL |

3. Create Google Slides certificate template with:

[[name]]
[[date]]

4. Open Extensions > Apps Script
5. Paste Code.gs
6. Update template URL
7. Run onOpen()

## Usage

Automation Menu → Email All Certificates
